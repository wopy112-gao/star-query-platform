v2.0.0 2026-05-15

🏗️ 重大架构变更：LLM 理解语义 + 三层缓存

移除「正则模板猜语义」的模糊模板，改为 LLM 真正理解自然语言后翻译 SQL， 配合三层缓存加速重复查询。

架构对比
旧架构（v1.x）： 用户 → 正则模板 → 猜语义 → 猜错就加补丁 → 无限循环 ↓ "感冒被当药品" → 加疾病库 "广州被当药品" → 加城市白名单 "下一个漏在哪" → 不知道 新架构（v2.0）： 用户 → [精确模板] → 命中 → 毫秒级返回 → [三层缓存] → 命中 → 毫秒级返回 → [LLM 翻译] → 写入缓存 → 正确SQL（仅首次）
