## 查询注意事项

| 注意点 | 说明 |
|--------|------|
| **JSON 字段解析** | `顾客点名药品`、`场景提及药品`、`订单药品`、`患者关键信息` 为 JSON 数组格式的 VARCHAR。查询单个药品用 DuckDB 的 `JSON_EXTRACT_STRING_ARRAY` 或 `list_aggr(json_extract_array(...), ...)` 等函数 |
| **字符串比较** | BIGINT 类型字段（场景ID、店员ID、门店ID）不能与空字符串 `''` 比较，应使用 `IS NULL` |
| **年龄字段** | 取值含 `'0'`（未识别）、`'-2'`（异常值）、数字字符串、年龄段字符串多种形态，过滤时注意 |
| **结果行数限制** | 正式环境 MAX_ROWS=500，系统自动追加 `LIMIT 500` |
| **仅支持 SELECT** | SqlValidator 拦截 DROP/INSERT/DELETE/UPDATE/ALTER/CREATE 等操作 |
